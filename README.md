# Quick Overview

NLP project consisting of a journaling demo that takes 1-5 paragraphs of first-person writing describing an emotional reaction to an event and predicts appraisal dimensions, expands emotion labels, and suggests a reframing prompt.

# Data

This project uses crowd-EnVENT, an event-based emotion/appraisal dataset introduced by Troiano et al. (2023).

# References

Troiano, E., et al. (2023). Dimensional Modeling of Emotions in Text with Appraisal Theories: Corpus Creation, Annotation Reliability, and Prediction. Computational Linguistics. MIT Press. https://doi.org/10.1162/coli_a_00461
